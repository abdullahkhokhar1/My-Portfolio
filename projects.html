Titanic Survival Prediction

Description:

Used the Titanic dataset to build a model predicting passenger survival. Here's a breakdown of the key steps:

  • • Data Visualization:
    • - Utilized libraries like Seaborn and Matplotlib for visualizing data.
    • - Gained insights into patterns and trends within the dataset.
  • • Data Cleaning:
    • - Addressed missing values in the 'Age' and 'Fare' columns by filling them with mean values.
    • - Eliminated the 'Cabin' column due to a high number of missing values.
  • • Feature Engineering:
    • - Transformed the 'Sex' column into numerical values (0 for female, 1 for male).
    • - Dropped less relevant columns such as 'Name', 'Ticket', and 'Embarked'.
  • • Model Building and Evaluation:
    • - Split the data into dependent and independent variables.
    • - Utilized 'train_test_split' to create separate training and testing datasets.
    • - Implemented Logistic Regression model for prediction.
Movie Rating Prediction

Description:

Developed a movie rating prediction model. Here's a breakdown of the key steps:

  • • Data Collection and Preparation:
    • - Loaded and merged datasets containing movies information, user ratings, and user profiles.
    • - Ensured data cleanliness by handling missing values in all relevant datasets.
  • • Exploratory Data Analysis:
    • - Leveraged powerful libraries like pandas, numpy, and seaborn for insightful data visualization.
    • - Explored correlations among features through heatmaps.
    • - Visualized user age distribution using bar plots to understand the dataset's characteristics.
  • • Feature Engineering:
    • - Selected key features, including "MovieID," "Age," and "Occupation," as independent variables.
    • - Transformed categorical data like "Gender" into numerical values for model compatibility.
  • • Model Building:
    • - Chose the Decision Tree Regressor for its suitability in predicting continuous numerical values.
    • - Used the "train_test_split" method to divide the data into training and testing sets.
    • - Trained the model on the training data, allowing it to learn patterns from the data.
  • • Model Evaluation:
    • - Assessed the model's performance using vital metrics: Mean Squared Error (MSE), Mean Absolute Error (MAE), and R-squared.
    • - These metrics quantified the model's accuracy in predicting movie ratings.
  • • Insights and Visualization:
    • - Visualized feature importances using a bar plot, highlighting the most influential features.
    • - Plotted a scatter plot comparing actual ratings with predicted ratings, offering an intuitive view of model performance.
